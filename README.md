# Plane Takeoff Animation ✈️

Simple airplane takeoff animation using OpenGL and Python.

## Features

* Simple runway animation
* Plane takeoff effect
* Adjustable speed using keyboard
* Automatic restart after plane leaves screen

## Controls

* **Right Arrow Key** → Increase speed
* **Left Arrow Key** → Decrease speed

## Requirements

Install OpenGL libraries:

```bash
pip install PyOpenGL PyOpenGL_accelerate
```

## Run the Project

Save the code as:

```bash
plane_animation.py
```

Run:

```bash
python plane_animation.py
```
